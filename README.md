## ✨ 主要功能

### 🚀 核心功能
- **文章管理**：支持 Markdown 富文本编辑，分类管理
- **资源中心**：整理分享学习资源、开发工具等
- **交互功能**：留言板、收藏系统和用户互动
- **热搜追踪**：实时显示热门话题和文章排行

### 🛒 特色模块
- **博客超市**：独特的资源分享平台（可替换为其他功能）
- **友情链接**：与其他博主互换链接，扩展社交网络
- **响应式设计**：适配手机、平板和桌面设备

### 🛠️ 管理功能
- 文章启用/禁用控制
- 评论审核系统
- 数据备份与恢复
- SEO 优化设置

## 🚀 快速开始

### 本地开发
```bash
# 克隆仓库
git clone https://github.com/yourusername/your-blog-repo.git

# 安装依赖
cd your-blog-repo
npm install

# 启动开发服务器
npm run dev
```

### 生产环境部署
```bash
# 构建生产版本
npm run build

# 启动服务
npm start
```

### Docker 部署
```bash
docker build -t my-blog .
docker run -d -p 8080:80 --name blog-container my-blog
```
